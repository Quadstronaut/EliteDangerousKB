# Refinery (summary)

- `bins` = number of different ore types you can refine **at once**; higher = less juggling.
- Max bins by class (A-rated): C1 4, C2 6, C3 8, C4 10.
- Bins rise with both class and rating; rating also sets power draw and cost.
