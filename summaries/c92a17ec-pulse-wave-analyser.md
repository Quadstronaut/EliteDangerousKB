# Pulse Wave Analyser (Coriolis module def)

Utility-mount mining scanner. Emits a pulse that briefly makes asteroids glow by mineral content,
revealing which rocks in a ring hold valuable cores/deposits before you commit to mining them.
Group `pwa`. Class **0** = mounts on a **utility slot**, not a hardpoint.

Key claims (from Coriolis JSON):
- **Class 0** (utility), ratings A–E. Mass 1.3 t. No damage.
- Power draw by rating: A 3.2 / B 1.6 / C 0.8 / D 0.4 / E 0.2 MW.
- Cost by rating: A 1,097,095 / B 365,698 / C 121,899 / D 40,633 / E 13,544 CR.
- Higher rating = wider/stronger pulse sweep. A-rated is the standard prospecting pick.
